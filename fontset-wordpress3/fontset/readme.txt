== Description ==

FontSet lets users increase, decrease or reset the font size of the page. The information is stored in a cookie so that there is no need for programming adjustments in the website.

FontSet is MVC compatible. If you have enough experience with the computing language PHP you can write your own layout mechanism (viewer).

You can find more information at the plugin's homepage http://www.creativepulse.gr/en/appstore/fontset

== Installation ==

Use the regular installation procedure to install this plugin.

That is, go to Admin > Plugins > Add New > Upload.

Alternatively you can upload it into the `/wordpress/wp-content/plugins/` directory
and then activate it in the Admin control panel for plugins.
